## 💡 How to Run

### Installation

#### 서버 시작하기

Backend config/.env
```bash
DJANGO SECRET KEY=
DJANGO_PASSWORD=
```

```bash
$ git clone https://github.com/2023-Summer-Bootcamp-Team-K/docker.git
$ cd docker/frontend
$ flutter pub get
$ flutter doctor
$ flutter run
```

#### 개발 서버 시작하기
```bash
$ git clone https://github.com/2023-Summer-Bootcamp-Team-K/docker.git
$ git submodule update --recursive --remote --init 
$ docker-compose -f docker-compose-dev.yml up -d
```
